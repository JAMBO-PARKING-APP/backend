## â“ FAQ

### Q: How do I know when an agent responds?
**A**: Messages refresh every 3 seconds. New agent messages appear automatically at the bottom of the chat.

### Q: Can I send images/files in chat?
**A**: Currently text-only. File attachments coming in next update!

### Q: What if my conversation gets closed?
**A**: You can still view the full history. To reopen, create a new conversation and reference the previous one.

### Q: Will chat drain my battery?
**A**: Minimal impact. Polling is only active when chat screen is open, ~3-5% battery per hour.

### Q: Can I switch languages during chat?
**A**: Yes! Language changes apply immediately to all screens including open chats.

### Q: What happens to my messages?
**A**: Messages are stored on secure servers. Marked as read once viewed. Deleted conversations lose history (backup recommended if needed).

### Q: How long for agent response?
**A**: Depends on availability. During business hours: 2-10 minutes. Off-hours: up to 24 hours.

### Q: Can I delete conversations?
**A**: Currently not deletable via app. Contact support to request deletion.

### Q: What's the category "Technical" for?
**A**: App crashes, bugs, technical issues. Use other categories for service-related issues.
